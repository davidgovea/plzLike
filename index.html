plzLike

A jQuery plugin for Facebook like-gates, backed by Firebase or custom endpoint.

    plzLike was created to make it easier to run like-based Facebook promotions (giveaways, contests, etc).

    A recent TOS change by Facebook marked an easing up on rules - the main point is now simply to display a "..not endorsed by.." statement.

    Firebase is a "real time back-end" allowing data storage and access from client-side code... with a great free tier. Using Firebase allows the plugin to be used from any page, collecting entrants without a back-end care in the world (not to mention the crazy real-time Firebase features that this project doesn't make use of)

    Features

    • Step users through "liking" your page and "entering" your promotion
    • Store entrants list in cloud storage
    • Configurable markup for the entry steps

    Usage

    1. Requirements

      1. Facebook App ID
      2. Facebook page to target (e.g. https://www.facebook.com/ Page Name )
      3. Firebase URL (sign up first)

    2. Include jQuery:

      <script src="http://ajax.googleapis.com/ajax/libs/jquery/2.0.0/jquery.min.js"></script>

    3. Data store:
      Sign up for Firebase and include firebase script:

      <!-- End of body, or somewhere in head if you must -->
<script type='text/javascript' src='https://cdn.firebase.com/v0/firebase.js'></script>

    4. Include plzLike:

      <!-- Up in da head -->
<link href="dist/jquery.plzlike.min.css" rel="stylesheet"/>

<!-- After jQuery -->
<script src="dist/jquery.plzlike.min.js"></script>

    5. Call the plugin:

      $("#element").plzLike({
    appId: <Facebook App ID>,
    page: <Facebook Page name>,
    firebaseUrl: <Firebase URL>
});
